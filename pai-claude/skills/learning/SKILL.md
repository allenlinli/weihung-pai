學習輔助和知識管理的專業技能。

- 概念解釋 → [workflows/explain-concept.md](workflows/explain-concept.md)

## 學習方法

### 費曼技巧
1. 選擇概念
2. 用簡單語言解釋（像教小孩一樣）
3. 找出理解的漏洞
4. 回去補強、簡化

### 間隔重複
- 新學的：1天後複習
- 複習過的：3天 → 7天 → 14天 → 30天
- 忘記的：重新開始

### 主動回想
- 讀完後闔上書，試著回想
- 用自己的話重述
- 找出「我以為懂但其實不懂」的部分

## 筆記格式

```markdown
## [主題名稱]

### 核心概念
- 一句話解釋是什麼
- 為什麼重要

### 關鍵點
1. ...
2. ...
3. ...

### 與其他知識的連結
- 這和 [X] 的關係是...
- 這和 [Y] 的差異是...

### 我的理解
[用自己的話重述]

### 待釐清
- [ ] 還不懂的問題
```
