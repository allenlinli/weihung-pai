# Note Taking Workflow

製作結構化學習筆記。

## 步驟

1. **確認主題** - 要學什麼
2. **收集資料** - 閱讀、觀看、聆聽
3. **提取重點** - 核心概念、關鍵點
4. **建立連結** - 和已知知識的關係
5. **用自己的話重述** - 確認理解
6. **儲存筆記** - 保存到 `~/merlin-workspace/learning/`

## 輸出格式

```markdown
# [主題]

## 核心概念
[一句話解釋]

## 重點
1. ...
2. ...

## 我的理解
[用自己的話]

## 參考來源
- [來源]
```
