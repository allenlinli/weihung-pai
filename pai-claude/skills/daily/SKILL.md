# Daily Skill

日常事務管理和規劃的專業技能。

## Workflow Routing

- 每日規劃 → [workflows/daily-plan.md](workflows/daily-plan.md)
- 週回顧 → [workflows/weekly-review.md](workflows/weekly-review.md)

## 事務管理

### 任務優先級
- **P0**: 今天必須完成
- **P1**: 本週內完成
- **P2**: 有空再做
- **P3**: 想法/靈感（可能不會做）

### 任務分類
- 🔴 工作
- 🟢 個人
- 🔵 學習
- 🟡 健康
- ⚪ 其他

## 每日流程

### 早上
1. 檢視今日任務
2. 確認優先級
3. 預估時間

### 晚上
1. 回顧完成情況
2. 未完成的移到明天或重新評估
3. 記錄阻礙和心得

## 待辦格式

```markdown
## [日期]

### P0 - 今日必做
- [ ] 任務1 (預估 30min)
- [ ] 任務2 (預估 1h)

### P1 - 本週
- [ ] 任務3

### 完成
- [x] 已完成的任務
```

## 資料儲存

所有日常記錄保存在 `~/merlin-workspace/daily/` 目錄。
