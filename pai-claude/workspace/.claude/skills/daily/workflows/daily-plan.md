# Daily Plan Workflow

每日規劃流程。

## 步驟

1. **回顧昨日** - 完成了什麼、未完成什麼
2. **列出今日任務** - 所有要做的事
3. **分配優先級** - P0/P1/P2
4. **預估時間** - 每個任務需要多久
5. **確認可行性** - 時間夠嗎？需要調整嗎？

## 輸出格式

```markdown
## [日期] 每日規劃

### 🎯 今日目標
[今天最重要的 1-3 件事]

### P0 - 必做
- [ ] 任務 (預估時間)

### P1 - 重要
- [ ] 任務 (預估時間)

### P2 - 有空再做
- [ ] 任務
```
