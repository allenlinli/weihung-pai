# Privacy Policy for Merlin Personal AI Assistant

**Last updated: January 2, 2025**

## Overview

Merlin is a personal AI assistant application that operates through Telegram Bot, powered by Claude AI, and integrates with Google services (Calendar, Drive, Gmail, Contacts) for private, personal use only.

## Data Collection

This application:
- Only accesses data from the authenticated user's own Google account
- Processes messages through Claude AI for generating responses
- May use voice transcription services (Google Gemini) for converting voice messages to text
- Stores conversation memory locally on the user's own server for context continuity
- Does not share any personal data with third parties beyond the AI services required for functionality
- Does not use any analytics or tracking services
- Stores OAuth tokens locally and securely on the user's own server

## Data Usage

### Telegram Messages
Your messages are processed by Claude AI to generate responses. Message content is not stored permanently except for optional long-term memory features you control.

### Voice Messages
Voice messages may be transcribed using Google Gemini API. Audio data is processed for transcription only and is not retained.

### Google API Data
Google API data is used exclusively to:
- Display and manage your calendar events
- Access and organize your Google Drive files
- Read and send emails on your behalf
- Search and manage your contacts

All data processing occurs locally on your private server.

## Data Sharing

We do not share any data with third parties beyond:
- Anthropic (Claude AI) for message processing
- Google (Gemini) for voice transcription (if enabled)
- Google APIs for calendar, drive, email, and contacts functionality

This is a personal-use application and no data is sold or shared for commercial purposes.

## Data Retention

- OAuth credentials are stored only as long as needed for the application to function
- Conversation memory can be cleared at any time through bot commands
- You can revoke access at any time through your Google Account settings

## Contact

For questions about this privacy policy, contact the repository owner.

## Changes

This privacy policy may be updated occasionally. Changes will be reflected in the "Last updated" date.
