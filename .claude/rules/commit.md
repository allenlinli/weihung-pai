# Git Commit 規範

## 訊息格式

- 使用 Conventional Commits：`type: description`
- 類型：`feat`, `fix`, `docs`, `refactor`, `test`, `chore`
- 訊息保持簡潔（50 字以內）

## 禁止事項

- 不加 `Co-Authored-By`
- 不加 `Generated with Claude Code`
- 不加 emoji

## 範例

```
feat: add permission request MCP tool
fix: resolve telegram auth issue
docs: update deployment instructions
```
